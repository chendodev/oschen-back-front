## 🚀 Instalación
1. Clona este proyecto.
2. Ve a la carpeta del proyecto
`cd windows`
3. Instala las dependencias
`npm install`
4. Corre el servidor Api Fake
`npm run fake-api`
5. Corre el ambiente local
`npm run start`
